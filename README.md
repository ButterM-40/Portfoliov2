# Portfolio v2

Personal portfolio site built with Next.js, deployed on Vercel.

## Content Update Workflow

1. Open `Admin.dc.html` in a browser and edit your content
2. Export from the Admin panel — it saves to `text.txt`
3. Run the sync script:
   ```
   npm run sync
   ```
4. Commit and push:
   ```
   git add src/data/portfolio.ts text.txt
   git commit -m "update content"
   git push
   ```

Vercel auto-deploys on push. Done.

## Dev

```
npm run dev
```

## Notes

- `text.txt` — JSON export from the Admin panel (source of edits)
- `src/data/portfolio.ts` — generated by `npm run sync`; do not edit by hand
- Base64 images in slides are stripped during sync; `image` fields stay empty until proper image hosting is added
